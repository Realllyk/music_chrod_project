# 2026-03-24 工作日志

## 完成事项

### 1. 录音采集流程简化
- 简化为 3 个按钮：开始录音、停止录音、保存录音
- API:
  - `POST /api/capture/start-recording` - 开始录音
  - `PUT /api/capture/stop-recording` - 停止录音
  - `POST /api/capture/save` - 保存录音
