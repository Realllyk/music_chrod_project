| 文件 | 动作 | 改动内容 |

## 2. API 契约

## 2.1 复用接口：歌曲详情

### `GET /api/songs/<song_id>`

**用途**
- 歌曲详情页加载基础信息
- 判断是否展示旋律入口
- 简谱页按需补充歌曲标题

**关键字段**

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | number | 歌曲 ID |
| `title` | string | 歌曲标题 |
| `artist_name` | string/null | 歌手名 |
| `melody_path` | string/null | 旋律结果路径 |
| `melody_url` | string/null | 旋律结果下载地址 |
| `status` | string | 当前状态 |

**前端使用规则**
- `details.html`：用 `melody_url || melody_path` 判断是否展示入口
- `melody.html`：若分析接口不返回标题，则再调用本接口补标题

## 2.2 新增依赖接口：旋律分析

### `GET /api/songs/<song_id>/melody-analysis`

**说明**
- 后端已明确：从 `songs_analysis` 表读取缓存
- 响应格式已统一为 `{ code, description, result }`

**用途**
- `melody.html` 的核心数据来源

### 响应结构

#### 成功响应

```json
{
  "code": 200,
  "description": "success",
  "result": {
    "song": {
      "id": 123,
      "melody_key": "melody:123"
    },
    "analysis": {
      "type": "melody",
      "midi_path": "/outputs/melody_123.mid",
      "key": {
        "tonic": "C",
        "mode": "major",
        "display": "C major",
        "confidence": 0.82
      },
      "time_signature": {
        "numerator": 4,
        "denominator": 4,
        "source": "midi_default"
      },
      "tempo": {
        "bpm": 112.3
      },
      "duration": {
        "seconds": 185.4
      },
      "notation": {
        "notes": [
          {
            "start": 0.52,
            "end": 0.78,
            "midi": 60,
            "pitch_name": "C4",
            "degree": "1",
            "octave_offset": 0,
            "duration_beats": 0.5
          }
        ],
        "measures": [
          {
            "index": 0,
            "start": 0.0,
            "end": 2.14,
            "degrees": ["1", "2", "3", "-"]
          }
        ]
      }
    }
  }
}
```

#### 错误响应示例

```json
{
  "code": 404,
  "description": "Melody analysis not found",
  "result": null
}
```

### 字段说明

#### 响应外层字段
| 字段 | 类型 | 前端用途 |
|---|---|---|
| `code` | number | 判断请求结果，`200` 表示成功，非 `200` 进入错误态 |
| `description` | string | 展示或记录错误说明 |
| `result` | object/null | 成功时为旋律分析数据，失败时为 `null` |

#### `result` 内字段
| 字段 | 类型 | 前端用途 |
|---|---|---|
| `song` | object | 当前歌曲信息，用于校验数据归属 |
| `analysis` | object | 旋律分析主体数据 |

#### `result.song`
| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | number | 当前歌曲 ID |
| `melody_key` | string | 当前旋律分析缓存键或标识 |

#### `result.analysis`
| 字段 | 类型 | 说明 |
|---|---|---|
| `type` | string | 分析类型，当前固定为 `melody` |
| `midi_path` | string | MIDI 文件路径，用于下载或展示附加信息 |
| `key` | object | 调性信息 |
| `time_signature` | object | 拍号信息 |
| `tempo` | object | 速度信息 |
| `duration` | object | 时长信息 |
| `notation` | object | 简谱渲染数据 |

#### `result.analysis.key`
| 字段 | 类型 | 说明 |
|---|---|---|
| `tonic` | string | 主音，如 `C` |
| `mode` | string | `major` / `minor` |
| `display` | string | 展示文案，如 `C major` |
| `confidence` | number | 调性估计置信度 |

#### `result.analysis.time_signature`
| 字段 | 类型 | 说明 |
|---|---|---|
| `numerator` | number | 分子 |
| `denominator` | number | 分母 |
| `source` | string | 来源，如 `midi_default` |

#### `result.analysis.tempo`
| 字段 | 类型 | 说明 |
|---|---|---|
| `bpm` | number | BPM 数值 |

#### `result.analysis.duration`
| 字段 | 类型 | 说明 |
|---|---|---|
| `seconds` | number | 总时长（秒） |

#### `result.analysis.notation.notes[]`
| 字段 | 类型 | 说明 |
|---|---|---|
| `start` | number | 起始秒数 |
| `end` | number | 结束秒数 |
| `midi` | number | MIDI 音高 |
| `pitch_name` | string | 绝对音名，如 `C4` |
| `degree` | string | 简谱级数，如 `1` / `#1` / `b3` |
| `octave_offset` | number | 八度偏移 |
| `duration_beats` | number | 拍长 |

#### `result.analysis.notation.measures[]`
| 字段 | 类型 | 说明 |
|---|---|---|
| `index` | number | 小节序号 |
| `start` | number | 小节起点秒数 |
| `end` | number | 小节终点秒数 |
| `degrees` | array<string> | 当前小节内的简谱级数 |

### 页面渲染约定
- 先判断外层 `code` 是否为 `200`
- 成功时从 `result.analysis` 读取 `key`、`time_signature`、`tempo`、`duration`、`notation` 并渲染页面
- 调性展示优先使用 `result.analysis.key.display`
- BPM 读取 `result.analysis.tempo.bpm`
- 时长读取 `result.analysis.duration.seconds`
- 主渲染数据从 `result.analysis.notation.measures` 读取
- hover 信息和兜底展示从 `result.analysis.notation.notes` 读取
- 升降号展示：`#1` / `b3`
- 八度展示：通过样式加上点/下点

### 异常约定

| 场景 | 前端表现 |
|---|---|
| `code=404, description=Melody analysis not found` | 显示“该歌曲尚未生成旋律结果” |
| 其他 `404` | 显示“歌曲不存在”或通用未找到提示 |
| `code=500` | 显示“旋律分析失败，请稍后重试” |
| `code=200` 但 `result.analysis.notation.notes` / `result.analysis.notation.measures` 为空 | 显示“暂无可展示的旋律数据” |
| 网络异常 | 显示错误态 + 重试按钮 |
