## 二、识别出的可复用组件/模块清单

### 2.1 已识别可复用模块

| 模块名称 | 功能 | 当前状态 | 建议 |
|----------|------|----------|------|
| **API 通信层** | 统一的 fetch 封装 | 分散在各页面 | 提取为 `js/api.js` |
| **基础样式** | body, card, btn, form-group | 各页面内联 | 提取为 `css/base.css` |
| **搜索列表组件** | 通用搜索 + 列表渲染 | 各页面重写 | 提取为 `js/components/search-list.js` |
| **表格组件** | 通用表格渲染 | recordings/capture 重复 | 提取为 `js/components/table.js` |
| **模态框组件** | 通用 Modal 弹窗 | capture/recordings 重复 | 提取为 `js/components/modal.js` |
| **表单组件** | 通用表单渲染 | songs/add, artists/add | 部分可复用 |
| **播放器组件** | 音频播放 | songs/detail | 可复用其他页面 |

### 2.2 推荐优先提取的模块

**优先级 P0（必须）：**
1. `css/base.css` - 统一基础样式（减少 ~500 行重复）
2. `js/api.js` - 统一 API 调用封装

**优先级 P1（推荐）：**
3. `js/components/search-list.js` - 通用搜索列表
4. `js/components/table.js` - 通用表格
